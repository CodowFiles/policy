# 2026年政策文件汇编

可多便利2026年市场开发部政策文件汇编系统。

## 功能特性

- 📚 **完整的政策文件库**：包含31份政策文件，按官方排序表整理
- 🔍 **全文搜索**：支持关键词搜索，快速定位政策内容
- 📱 **响应式设计**：完美适配桌面端、平板和移动设备
- 📝 **双版本查看**：支持文字版和扫描版PDF切换
- 🚀 **GitHub Pages部署**：支持分布式部署，随时访问

## 本地使用

直接在浏览器中打开 `index.html` 即可使用。

## 部署到GitHub Pages

1. 在GitHub创建新仓库
2. 将本项目代码推送到仓库
3. 进入仓库 Settings -> Pages
4. Source 选择 "GitHub Actions"
5. 访问自动生成的链接即可
